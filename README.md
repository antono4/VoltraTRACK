# VoltraTRACK ⚡

> **Created by Antono**


![VoltraTRACK Preview](hero-car.jpg)

Electric vehicle tracking and management system.

## ✨ Features

- 📍 Real-time vehicle tracking
- 🔋 Battery monitoring
- 🗺️ Route history
- 📊 Fleet management
- 🔔 Alert notifications

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Leaflet Maps

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/antono4/VoltraTRACK.git

# Open in browser
cd VoltraTRACK
open index.html
```

## 📁 Project Structure

```
VoltraTRACK/
├── index.html      # Main page
├── css/            # Stylesheets
├── js/             # JavaScript files
└── assets/         # Static assets
```
